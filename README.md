## Key Projects

### Quantitative Finance & ML
*   **[Deeplearning-option](https://github.com/MeilinP/Deeplearning-option)**: End-to-end deep learning framework for options trading (Linear, MLP, CNN, LSTM) trained on Polygon.io data.
*   **[Market Regime Identifier](https://github.com/MeilinP/market-regime-identifier)**: Trading signal system using HMM and volatility clustering to detect market regimes.
*   **[Market Tempo](https://github.com/MeilinP/market-tempo)**: Exploratory analysis correlating jazz musical patterns (Charlie Parker) with S&P 500 volatility.
*   **[FinBERT Sentiment Analysis](https://github.com/MeilinP/FinBERT_Sentiment_Analysis_on_SP500Constitutes)**: NLP-driven sentiment analysis on S&P 500 constituents using FinBERT.
